# AGENTS.md — MOABOM 작업 지침

이 폴더는 **MOABOM 대시보드 웹앱** 전용 작업 공간이다.

## 프로젝트 목표
- 텔레그램으로 받는 구직 요약을 웹에서 보기 좋게 탐색/관리
- 핵심 기능: 목록, 필터, 북마크, 제외, 메모, 플랫폼별 요약
- 배포 목표: Vercel

## 우선 원칙
1. **작게 만들고 빨리 검증**: 먼저 MVP를 완성한다.
2. **데이터 신뢰성 우선**: 링크 유효성/중복 제거를 기본 체크한다.
3. **UI는 단순·빠름**: 모바일 우선, 한 화면에서 핵심 작업 가능.
4. **변경은 기록**: 중요한 결정은 `Planning.md`에 즉시 반영.

## 기술 기본안
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- 데이터 저장: 초기 JSON/SQLite → 이후 Supabase 전환 가능
- 배포: Vercel

## 작업 방식
- 큰 기능은 반드시 작은 단위로 쪼개서 커밋
- TODO는 `Planning.md` 체크박스로 관리
- 새 이슈 발견 시:
  1) 재현 방법 기록
  2) 영향 범위 기록
  3) 임시 우회/근본 해결 분리

## 품질 체크리스트 (머지 전)
- [ ] `npm run lint` 통과
- [ ] `npm run build` 통과
- [ ] 핵심 페이지 모바일 뷰 확인
- [ ] 링크 클릭/상태 변경(북마크, 제외) 동작 확인
- [ ] 빈 상태/오류 상태 UI 확인

## 보안/운영
- 비밀값은 `.env.local`에만 저장, 저장소 커밋 금지
- 외부 API 키는 최소 권한 사용
- 로그에 개인정보/토큰 남기지 않기

## 완료 기준 (MVP)
- 목록/필터/상태 변경/요약 카드가 동작
- 기본 데이터 입력 경로 확보(수동 업로드 또는 스크립트)
- Vercel 프로덕션 URL에서 정상 동작
